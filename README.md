# GuessTheNumber

A simple and fun number guessing game written in Java.  

## Description
Guess the number randomly generated between **1 and 10**.  
You get **only two attempts**, and hints will tell you if your guess is too high or too low.  
This is a beginner-friendly project to practice Java basics like loops, input handling, and random numbers.  

## How to Play
1. Run the program in your IDE or terminal.  
2. Enter `'y'` to start the game.  
3. Enter `'r'` to generate a random number.  
4. Guess the number! Hints will guide you.  
5. Try to win in **two attempts**.  

## Features
- Random number generation between 1 and 10  
- Input validation (only numbers allowed)  
- Hints for high or low guesses  
- Beginner-friendly  

## Usage
Compile and run the program:

```bash
javac GuessTheNumber.java
java GuessTheNumber
